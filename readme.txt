############################# Instructions to run the code #################

1. Place all the files: train.csv , evaluation.csv and python code file product_classification.py in your current working directory
2. Important python librarires like numy, pandas, sklearn , nltk should be installed on your system. 
3. Run python_classification.py for feature extraction, model training and model prediction.
4. The output file submissions.csv gets generated in the same working directory.
